lcrt - Lynx Certificate

The Lynx (and later ocelot) are parts of the SEP. This file is a IMG4 signed copy of that certificate. Along with some other structured data.

trust-object

Trust object (current and specified) are ASN.1 files encoding the trusted TLS certificate for when the FDR process contacts out to the apple server to restore factory data (things like serial number etc). A restore machine is expected to provide a FDR proxy during the restore process, this allow the device to connect out to download data directly to Apple. These certificates are rooted out of FDR-DC-SSL-ROOT

ASN.1 Sequence

Data file magic secb

Sequence staring with trst which is the trust object. Sequence next is rssl which is the transport root Sequence final is rvok which is revoked certificates
